Что такое псевдо-классы?

Псевдоклассы определяют динамическое состояние элементов,
которое изменяется с помощью действий пользователя, а также положение
в дереве документа. Примером такого состояния служит текстовая
ссылка, которая меняет свой цвет при наведении на неё курсора мыши. При использовании
псевдоклассов браузер не перегружает текущий документ,
поэтому с помощью псевдоклассов можно получить разные динамические эффекты на странице.
Синтаксис применения псевдоклассов следующий:
Селектор:Псевдокласс {
Описание правил стиля
}

Hover

Наверное большинство видели сайты,
где при наведении на что-то с этим обьектом что то происходит.
Некоторые задумывались:《как это сделано?》

В этом помогает самый популярный псевдокласс:
Hover.
И так, как он работает?

Сначала создоём селектор класса, например, button:

.button {
background-color: black;
color: white;
width: 250px;
height: 100px;
}

Дальше мы с новой строки пишем название нашего селектора,
двоеточие(:),
и пишем hover,

А после открывающей фигурной скобки начинаем перечислять свойства,
которые должны изменится.
Например, вот так:

.button:hover {
width: 300px;
background-color: white;
color: black;
transition: 3s;
}

Что же мы сделали? Мы увеличили кнопку до 300px,
изменили цвет кнолпки на чёрный,
а цвет текста на белый за 3 секунды.

Время, за которое будет изменятся значение, отвечает свойство transition.

Before и after

Это, скорее, селекторы для удобства

У нас есть список:

  1. Пункты
  2. Пункты
  3. Пункты
  4. Пункты
  5. Пункты
  6. Пункты

И мы хотим что-бы перед
каждым пунктом стоял символ "《",
а после каждого пункта стоял символ "》".

В этом поможет псевдоклассы before и after.
Вы скажите: а зачем, если можно
вручную это всё сделать. Да,
можно, но тогда это займёт больше времени и если у
нас будет, например, 100 пунктов, то Вам
будет вразы сложнее
и дольше єто делать, да и код будет менее читаимый.

Синтаксис

Наши псевдоклассы не будет работать без свойства content: 'что мы хотим добавить(в нашем случаи
у псевдокласса before будет значение 《, а у after 》'.

Синтаксис такой-же, как и у hover, только мы ствавим два двоеточия,
и пишем следуещее:
.ol {
font-size: 40px;
.ol::before {
content:'《';
}
.ol::after {
content: '》';
}

Visited

Вы создали сайт с кучай ссылок. Пользователь хочет посмотреть все эти ссылки, но их так много, что он по случайности нажимает на одну и ту-же ссылку. Как этого избежать? Псевдокласс visited нам поможет!. Он работает аналогично
своим "братьев". Только он срабатывает после нажатия на ссылку и перехода по ней, а когда
пользователь перейдёт по ссылке и вернётся обратно, ссылка изменится, и пользователь больше не ошибётся!
Как пользоватся этим псевдоклассом - сейчас узнаем

Синтаксис у него такоей-же, как и у hover. Он ничем от него не отличается, только срабатывают они в разных случаях. У нас есть ссылка-якорь, которая ведёт нас в самый верх сайта

Дальше, мы даём ей класс a(все стили я писать не буду, покажу только с visited)

Пример:

a:visited {
color: blue;
}

CSS transform

Вращать, двигать, увеличивать, искожать - всё это и много другое умеет свойство transform! Как оно работает? Сейчас узнаем

Для наглядности все свойства будут срабатывать при наведении. Но естественно, можно это делать и без hover

Translate

Описание:
перемещает обьекты по укакзанам координатам
Синтаксис:
transform: translate(*координаты по оси X*, *координаты по оси Y*);

Что-бы переместить влево, задавайте отрицательное значение координатам по оси X,
что-бы переместить влево,
задавайте отрицательное значение координатам по оси Y
TRANSLATE

Rotate

Описание:
переварачивает обьект
на заданое количество градусов(deg)
Синтаксис:
transform: rotate(90deg);

Что-бы повернуть против часовой стрелки, задавайте отрицательное значение
ROTATE

Scale

Увеличивает обьект относительно его центра в заданое количество раз
Синтаксис: transform: scale(1.2);

Что-бы уменшить, пишите 0.9, например, и наш блок уменшится на 0.10
